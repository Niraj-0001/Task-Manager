How to Install

Since this application is not distributed via the Google Play Store, you can easily sideload the APK by following these 3 simple steps:

1

Download the APK

Tap the main Download button above to save the app-release.apk installer to your Android device.

2

Enable Unknown Sources

When opening the file, if prompted, select Settings and toggle on the option to "Allow installation from this source".

3

Install and Launch!

Go back to the prompt, tap Install, then open your brand new luxury study companion from your apps list.
